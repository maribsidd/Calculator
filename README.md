Simple Calculator:
A command-line calculator that performs basic arithmetic operations including addition, subtraction, multiplication, and division.

Description:
This is a Python-based calculator program that provides a simple menu-driven interface for performing mathematical calculations. The program runs in a continuous loop until the user chooses to exit. This project demonstrates fundamental programming concepts including conditional statements, loops, input validation, and error handling. It's perfect for beginners learning Python or as a portfolio project.

Features:
Four basic arithmetic operations (addition, subtraction, multiplication, division)
Continuous operation with menu-driven interface
Input validation for operation choices
Division by zero error handling
Clean and simple code structure
No external dependencies required
User-friendly output formatting

How to Play:
1. Run the program
2. Select operation (1-5) from the menu
3. Enter two numbers when prompted
4. View the calculated result
5. Choose another operation or exit (option 5)

Variables Used:
ch - User's choice of operation
n1 - First number input
n2 - Second number input
r - Result of calculation

Main Components:
1. Welcome Message - Display calculator title
2. Menu Display - Show available operations
3. Input Collection - Get operation choice and numbers
4. Calculation - Perform selected arithmetic operation
5. Error Handling - Handle division by zero
6. Results Display - Show calculation result
